# Changelogs

## Announcements: v0.1.0 - 2026-06-12

- **Interactive Editing Foundation:** Archy AI now supports direct room editing, geometry changes, room add/delete actions, opening controls, undo/redo history, and clear invalid-edit feedback.
- **Professional Delivery Flow:** This release adds branded PDF/PNG exports, export templates, secure client preview links, and saved-plan version history for a more complete client handoff workflow.
- **Presentation-Ready Export Quality:** Standard, High Resolution, and Print export modes now give client handoffs sharper PNGs, larger PDF sheets, and richer presentation metadata.
- **Concept-Ready Interiors and Exteriors:** Furniture suggestions, material presets, manual palette editing, saved interior concept briefs, and exterior facade concepts now make generated plans more useful for client presentation and future rendering workflows.
