# Latest stable release: v0.1.0

Released: June 12, 2026

For most users, the latest stable release is the recommended release.

## Highlights

- **Interactive Floor Plan Editing:** Added room selection, label and type editing, direct room dragging and resizing, room add/delete controls, and undo/redo history with keyboard shortcuts.
- **Professional Client Outputs:** Added branded PDF and PNG exports with title blocks, room schedules, legends, scale bars, dimensions, doors, windows, and selectable Presentation, Technical, and Real Estate templates.
- **High-Resolution Exports:** Added Standard, High Resolution, and Print quality options for larger PNG outputs and print-ready PDF sheets.
- **Shareable Client Previews:** Added secure public share links with owner controls to create, copy, regenerate, and revoke client preview access.
- **Deterministic Design Intelligence:** Expanded residential templates, opening regeneration, furniture suggestions, and material presets so generated plans are more complete and consistent.
- **AI Room Specifications:** The AI now returns per-room area, aspect ratio, zone, and adjacency hints, enabling proportional room scaling and smarter door/window placement.
- **Interior Concept Briefs:** Added room-level concept cards with style presets, palette cues, furniture direction, and render-ready prompts for future image workflows.
- **Exterior Facade Concepts:** Added whole-plan facade concept briefs with style presets, material cues, massing notes, roofline direction, and render prompts for future image generation.
- **Role-Based Product Flow:** Added signup user segmentation and role-specific starter briefs for homeowners, architects/designers, and real estate/builders.

## What's Changed

- Added signup user segmentation for Homeowner, Architect / Designer, and Real Estate / Builder users.
- Added deterministic AI floor plan generation, refinement support, and expanded residential template variants.
- Added room editing, room geometry editing, room count editing, opening editing, invalid edit feedback, and editing history.
- Added professional PDF and PNG export with dimensions, scale bars, room schedules, and multiple export templates.
- Added high-resolution and print export quality options with richer presentation metadata.
- Added secure shareable client preview links and saved-plan version history.
- Added opening regeneration, furniture placement suggestions, room materials, finishes, and manual material selection.
- Added interior concept briefs with selected-room style presets, shared preview cards, and export summaries.
- Added exterior facade concept briefs with facade style presets, shared preview card, and export summaries.
- Added AI room specifications with per-room area, aspect ratio, zone, and adjacency hints for proportional room scaling.
- Changed canvas background from line grid to dotted grid pattern.
- Changed default theme from dark to light.
- Added theme toggle (light/dark mode switcher) to header and editor.
- Updated dashboard and editor starter briefs to use role-specific content.
