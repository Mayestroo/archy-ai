# Preview release: v0.1.0

Released: June 12, 2026

## Unreleased

### Editor Q&A Onboarding Wizard

The editor now starts with a guided Q&A wizard — answer step-by-step questions about floors, area, shape, rooms, and extras, then review and generate. Canvas progress overlay shows completed steps and animated generation stages. Shape and room modals provide visual selection.

### Post-Generation Tools

- **Style Picker Modal:** Searchable style card grid with interior/exterior scope toggle. Connects to existing concept brief creation.
- **Finish Picker Modal:** Material/Colour tabs with category filter, palette card grid, and color swatches. Applies finishes to selected rooms.
- **Furniture Browser:** Hierarchical category browser with 7 furniture groups. Click an item to add the corresponding room to the plan.
- **Render Scene Panel:** Camera preview stub, FOV slider, aspect ratio selector, render prompt input, and disabled render button in the Properties sidebar.

---

There is no separate preview release channel yet. Current preview-facing changes
are tracked in the latest stable release notes.

## Highlights

- **Interactive Floor Plan Editing:** Added the first complete editing loop for rooms, openings, geometry, history, and validation feedback.
- **Client Sharing and Export:** Added public preview links plus branded PDF and PNG exports for stakeholder review.
- **High-Resolution Exports:** Added quality controls for Standard, High Resolution, and Print client outputs.
- **Generated Plan Enrichment:** Added deterministic openings, furniture, materials, finishes, and expanded residential templates.
- **Interior Concept Briefs:** Added selected-room concept briefs with style presets and render-ready prompts for future image generation.
- **Exterior Facade Concepts:** Added whole-plan facade concept briefs with style presets, material cues, and render prompts.
- **AI Room Specifications:** Added per-room area, aspect ratio, zone, and adjacency hints for proportional room scaling and smarter placement.
